Pour démarrer le projet : 
1 - cd ./api
1 - Mettre la configuration de sa base de donnée dans ./ormconfig.json
2 - yarn start pour démarrer l'api
3 - cd ../
5 - yarn dev pour démarrer la todo en mode développement

Utilisation :

Sur la gauche, liste des todos, "en cours" en rouge est cliquable, passage en mode terminé.

Bouton modifié pour modifié sa todos

Sur la droite, deux formulaire, un pour créer une todo et l'autre pour filtrer les résultats
